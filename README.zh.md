# dashi-skills

[limerick](https://github.com/limerickgds) 的个人 skills 仓库，旨在帮助 AI 助手更好地了解人，并具备自我成长和持续优化的能力。

## 目标

- 帮助 AI 助手深入理解个人用户的偏好与习惯
- 让 AI 能够通过自我反思与优化持续进化
- 提供可复用的 skill 模块，增强 AI 与人的协作效率

## 当前 Skill 列表

| Skill | 说明 |
|-------|------|
| `socratic-mirror` | 通过苏格拉底式提问与逻辑审视，帮助用户检查假设、盲点与推理模式。 |
| `execution-reviewer` | 针对已完成的任务回顾执行过程，整理成简洁的复盘、交接说明或改进备忘录。 |

## 使用方式

从这个仓库安装 skills：

```bash
npx skills limerickgds/dashi-skills
```

这条命令会安装 `limerickgds/dashi-skills` 发布出来的 skills。
