# dashi-skills

A personal skills repository by [limerick](https://github.com/limerickgds), designed to help AI assistants better understand people and develop the ability to self-grow and self-optimize.

## Purpose

- Help AI assistants build a deeper understanding of the individual user
- Enable AI to continuously improve and adapt through self-reflection and optimization
- Provide reusable skill modules that enhance AI-human collaboration

## Usage

Install skills locally in your project:

```bash
npx skills add <skill-package>
```
